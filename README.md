# Генератор документов по 152-ФЗ

Веб-сервис для автоматической генерации документов, необходимых для соответствия сайта требованиям Федерального закона № 152-ФЗ «О персональных данных».

## Возможности

Сервис генерирует следующие документы:
- 📄 Политика обработки персональных данных
- 🍪 Политика использования cookie-файлов
- ✅ Согласие на обработку персональных данных
- 📧 Согласие на рассылку
- 🔄 Согласие на распространение ПДн
- 📋 Пользовательское соглашение

## Установка и запуск

### Требования
- Python 3.7+
- Flask
- python-docx

### Установка зависимостей

```bash
pip install flask python-docx
```

### Запуск сервера

```bash
cd /workspace/fz152-doc-generator
python app.py
```

Сервер запустится на адресе: `http://localhost:5000`

## Как использовать

1. Откройте браузер и перейдите на `http://localhost:5000`
2. Выберите тип документа из списка
3. Заполните данные вашей организации:
   - Наименование (ООО, ИП и т.д.)
   - ИНН
   - Юридический адрес
   - Email для связи
   - Телефон
   - Адрес сайта
4. Отметьте функции, которые используются на вашем сайте:
   - Формы сбора данных
   - Системы аналитики
   - Cookie-файлы
   - Email-рассылка
   - Передача данных третьим лицам
   - CRM-системы
   - Платежные системы
   - Социальные виджеты
5. Укажите, какие персональные данные вы собираете
6. Нажмите кнопку «Сгенерировать и скачать документ»
7. Получите готовый документ в формате .docx
